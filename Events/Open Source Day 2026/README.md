# Open Source Day 2026

## Overview

Open Source Day consisted of two technical sessions and one special attraction of *Reverse Engineering*.
The event focused on **Open Source fundamentals**, **Git & GitHub**, and **Cybersecurity concepts**, covering theoretical and practical foundations along with real-world security scenarios.

---

## Session Details

#### **Session 1: Version Control Systems & GitHub**

* Open Source basics and Version Control Systems
* Centralized vs Distributed VCS
* Essential Linux commands like: `cd`, `ls`, `pwd`, `mkdir`, `touch`, `cat`
* **Git**: Installation, basic commands: `init`, `status`, `add`, `commit`, `log`, `push`, `pull`
  * Branching, merging and advance Git commands: `revert`, `reset`, `rebase`
* **GitHub**: Repository setup, Pull requests and Open source contribution process

---

#### **Session 2: Cybersecurity**

* Introduction to Cybersecurity and attack lifecycle
* Common attack types and CIA Triad
* Cybersecurity tools overview: `John the Ripper`, `Hashcat`, `Nmap`, `Maltego`, `Wireshark`
* Security precautions and detection techniques

---

#### **Special Attraction: Reverse Engineering**

* Introduction to Reverse Engineering
* APK build lifecycle and JADX demonstration
* Risks of hardcoded secrets
* Secure design practices and ethical considerations
